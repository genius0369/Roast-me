Roast Battle Arena is a fun, interactive web app built using HTML,
CSS, and JavaScript.

It includes: - Multiplayer Roast Battle Mode - AI-Style Dynamic Roast
Generator - 3D Neon UI Design - Sound Toggle - Persistent Leaderboard
(LocalStorage)






This project runs entirely in the browser. No backend required.

------------------------------------------------------------------------

## Features Overview

### Multiplayer Roast Battle Mode

-   Two players enter their names
-   Each receives a dynamic AI-style roast
-   Random scoring system decides winner
-   Leaderboard tracks total wins
-   Data stored using LocalStorage

### AI Roast Engine

-   Name-based dynamic roast generation
-   Adjustable intensity slider (1--5)
-   Smart trait + ending combination logic
-   Higher intensity adds extra spice

### 3D Neon UI

-   Perspective transform effects
-   Neon glow box shadows
-   Animated depth hover effect
-   Dark cyber-style background

### Sound Effects

-   Roast sound on action
-   Toggle sound ON/OFF

### Leaderboard System

-   Tracks battle wins
-   Sorts players by highest wins
-   Automatically persists in browser
